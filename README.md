# Documentacion de aplicacion

## Objetivo
### El objetivo principal de una aplicación nutritiva es ayudar a las personas a mejorar y mantener sus hábitos alimenticios de manera saludable, proporcionando información personalizada sobre su dieta, control de consumo de nutrientes y recomendaciones basadas en sus necesidades, metas y estilo de vida.
